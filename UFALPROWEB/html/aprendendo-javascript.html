O que é Javascript?

JavaScript é uma das linguagens de programação mais populares e usadas no mundo. Hoje você vai aprender mais sobre o que ela é, como funciona, onde pode ser aplicada e como ela se diferencia dos demais tipos.

Também conhecida apenas como JS, JavaScript é uma linguagem de programação usada para desenvolver aplicações, sistemas e serviços de alta complexidade.Com ela, você pode criar páginas web dinâmicas, animações, mapas interativos, gráficos em três dimensões, aplicativos para dispositivos móveis e games para plataformas portáteis.

Ela atua como um complemento às linguagens HTML, CSS e PHP no momento em que o desenvolvedor vai construir uma página na internet. Resumidamente, estas linguagens de programação interagem entre si assim:

  • HTML – Permite adicionar conteúdos em uma página através de marcação de textos. É a estrutura óssea de um site.
  • CSS– Especifica o estilo do design do layout de uma página. É tudo relacionado à aparência de um site.
  • PHP – Dinamiza o processo de desenvolvimento de uma página adicionando extensões e aplicações dinâmicas.
  • JavaScript – Controla o comportamento dos elementos de uma página, executando funções dinâmicas dos conteúdos, animações, aplicativos e interações em geral.

Neste caso, o HTML seria a estrutura geral, as portas, corredores e paredes. O CSS entra aqui como a decoração geral, os tapetes e tintura usados para deixá-la mais bonita. E por fim, o JavaScript adiciona a responsividade e a funcionalidade da casa, em que você pode abrir portas, ligar ou desligar a luz por interruptores. Você até pode gerenciar tudo isso sem JS, mas é como se estivesse atrasado algumas décadas.

Escreva seu primeiro código em JavaScript!

Para começar, mostraremos como adicionar alguns comandos JavaScript básicos à sua página, criando um exemplo de "Olá mundo!" e mostrando esse exemplo no console do navegador. Sendo assim vamos assumir que você tem um conhecimento básico/iniciante de HTML e CSS e conhece um pouco sobre linguagens de programação para este teste. Mas se não souber tudo bem ao final do post deixaremos uma grande dica para se aprofundar nos estudos. Agora vamos lá!

Primeiro, vá para o seu site de teste e crie uma nova pasta chamada scripts. Em seguida, dentro da nova pasta de scripts que você acabou de criar, crie um novo arquivo chamado main.js. Salve na sua pasta de scripts.

Em seguida, no seu arquivo index.html, insira o seguinte elemento em uma nova linha logo antes da tag de fechamento ‹/body›:

‹script src="scripts/main.js"›‹/script›

Isso é basicamente a mesma coisa que o elemento ‹link› para o CSS — ele aplica o JavaScript à página, para que ele tenha efeito no HTML (junto com o CSS e qualquer outra coisa na página).

Agora vamos criar uma variável. Mas o que é uma variável? Bom, de maneira geral podemos dizer que variável é um espaço na memória do computador onde um determinado dado é guardado. São como caixinhas onde você pode guardar dados. Em javascript elas podem ser declaradas com as palavras reservadas let, var e const .

Neste exemplo vamos utilizar o const, que serve para declarar constantes, aqui, como o valor não será alterado, constantes são mais indicadas. Vamos ao código!

Em seu arquivo main.js escreva:

const frase = "Hello World!"

Com isso declaramos uma variável de nome frase que recebe uma string Hello World! A declaração de strings em JavaScript podem serem feitas com aspas duplas ou simples.

Logo em seguida, vamos mostrar esse valor contido na variável palavra. Sendo assim escreva:

console.log(frase)

Após isso vá ao seu navegador com o arquivo index.html aberto, clique com botão direito do mouse e abra o inspecionar elemento. Após isso clique em console e se você fez todos os passos certinho, a palavra Hello World! deve estar aparecendo no console. Simples Não?

Outra forma seria simplesmente passando o "Hello World!" dentro do console.log. Ficaria assim:

console.log("Hello World!")

Como somar dois números?

Outra coisa muito comum quando se está programando, é fazer operaçãoes matemáticas. Sendo assim, vamos agora aprender a como fazer essas operaçãoes em javascript.

Para se fazer cálculos matemáticos em javascript é muito semelhante a forma como você normalmente as faz em uma calculadora, basicamente você atribui a operação desejada (multiplicação, divisão, adição, subtração, etc.) a uma variavel e manda exibir no console, ou simplesmente faz todo o cáculo dentro do próprio console. Nesse exemplo mostraremos como armazenar em uma constante.

Sendo assim, primeiro de tudo, vamos pegar o conceito aprendido anteriormente e declarar uma constante que vai receber o valor dessa operação:

const calculo = 1 + 1

Apos isso apenas precisamos passar a constante para o console:

console.log(calculo)

E após isso verificar o resultado no console do navegador, se estiver tudo correto, você receberá o valor da soma, ou seja 1 + 1 = 2.

Como juntar e exibir texto e valores de variáveis?

Se já chegou até aqui significa que está gostando do javascript. Ficamos muito felizes com isso. Agora vamos untar texto e valores de uma variável e exibir isso pra gente no console do navegador.

isso que iremos fazer agora é um processo, que em programação chamamos de Concatenar. Mas o que é isso? Concatenar é uma palavra chique da programação que significa "colocar junto". Para colocar strings juntas em JavaScript, usamos o operador (+), o mesmo usamos para adicionar números, mas neste contexto é algo diferente.

Mas... Por que é diferente? Bom mesmo que estejamos passando uma váriavel que contém numeros dentro dela, quando usamos operador (+) para juntar um texto e um numero, o valor que será exibido ou armazenado na variável será uma string. Existem muitas formas de se concaternar em javascript, seja utilizandoas funções  CONCATENATE ou CONCAT,TEXT ou TEXTJOIN, mas nesse exemplo utilizaremos o operador (+) mesmo.

Vamos lá! Primeiro declare uma constante que irá receber a soma de dois números, nestes exemplo 5 + 8.

const somando = 5 + 8

Após isso, chame o console.log inserindo o texto que deseja exibir entre áspas e logo em seguida adicione o operador (+) e por fim o nome da constante que deseja exibir no caso do nosso exemplo a constante somando.

console.log("O resultado da soma é = " + somando)

Se você fez tudo certinho, o resultado obtido será: O resultado da soma é = 13.
